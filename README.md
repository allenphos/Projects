# Data Science and Machine Learning Projects

## Overview
This repository contains selected projects completed during my studies at Noroff University and [Data Loves Academy](https://www.linkedin.com/company/data-loves-academy/posts/?feedView=all), with a focus on building practical skills in data science, machine learning, and deep learning.

The projects demonstrate hands-on experience with data preprocessing, exploratory analysis, regression, classification, class imbalance handling, regularization, and convolutional neural networks using Python.


## Projects

1. **Computer Vision with PyTorch**  
   Built a convolutional neural network in PyTorch for handwritten digit classification on the MNIST dataset. The project included dataset exploration, image visualization, batch preparation with DataLoaders, implementation of a TinyVGG-inspired CNN, and training on both CPU and GPU. Model performance was evaluated through sample predictions, misclassified examples, and a confusion matrix. The analysis showed that the model most frequently confused digits such as **7 and 2**, and **6 and 0**, reflecting the visual similarity of some handwritten samples. The impact of different convolution kernel sizes on output feature map dimensions was also explored.  
   [Link to the notebook](https://colab.research.google.com/drive/1504iTP81TyoWa3wDFEzoDqxE2TiR_HzD#scrollTo=-FeYlGobpYBe)

2. **Car Price Prediction**  
   Developed a multiple linear regression model to predict car prices based on features such as mileage, engine size, and brand. The project included data cleaning, exploratory visualization, and model evaluation.

3. **Credit Data Preprocessing**  
   Prepared a credit application dataset for downstream analysis and modeling. The workflow included handling missing values, transforming features, and treating outliers to improve data quality and support more reliable credit risk assessment.

4. **Customer Segmentation using Machine Learning**  
   Built a customer segmentation workflow using logistic regression for targeted marketing in the automobile industry. Class imbalance was addressed with SMOTE and SMOTE-Tomek, and the project included preprocessing, model training, and evaluation.

5. **Overfitting and Regularization in Regression**  
   Compared Linear Regression and Polynomial Regression models to demonstrate overfitting in predictive modeling. Applied Ridge, Lasso, and ElasticNet regularization techniques to improve generalization and reduce model complexity.


## Bank Customer Churn Prediction
The projects in the **Bank Customer Churn Prediction** directory were completed as part of participation in the [Bank Customer Churn Prediction Kaggle competition](https://www.kaggle.com/competitions/bank-customer-churn-prediction-dlu-course-c-2).

1. **Logistic Regression with Scikit-Learn**  
   Built a logistic regression model to predict customer churn using demographic, financial, and behavioral features. The project included preprocessing, exploratory data analysis, model training, and evaluation.

2. **Polynomial Feature Engineering for Enhanced Churn Modeling**  
   Extended the baseline logistic regression approach with polynomial feature engineering to improve churn prediction performance. The workflow included preprocessing, feature engineering, model training, and comparative evaluation across multiple polynomial degrees.

3. **Decision Tree Classifier for Bank Customer Churn Prediction**  
   Developed a decision tree classifier for customer churn prediction and explored methods to improve performance under class imbalance. The project included preprocessing, feature selection, model training, evaluation, and experimentation with class weighting, resampling, and tree depth tuning.
